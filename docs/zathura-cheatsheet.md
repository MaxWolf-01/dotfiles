# Zathura Cheatsheet

## Navigation

| Key | Action |
|-----|--------|
| `j` / `k` | Scroll down / up |
| `h` / `l` | Scroll left / right |
| `^d` / `^u` | Half-page down / up |
| `^f` / `^b` | Full-page down / up |
| `Space` / `,` | Page forward / back |
| `J` / `PgDn` | Next page |
| `K` / `PgUp` | Previous page |
| `gg` | First page |
| `G` | Last page |
| `nG` | Go to page n |
| `P` | Snap to current page |
| `H` / `L` | Top / bottom of current page |

## Jump List

| Key | Action |
|-----|--------|
| `^o` / `^i` | Jump backward / forward |
| `^j` / `^k` | Bisect between last two jump points |

## Search

| Key | Action |
|-----|--------|
| `/` / `?` | Search forward / backward |
| `n` / `N` | Next / previous result |

## Zoom

| Key | Action |
|-----|--------|
| `+` / `-` | Zoom in / out |
| `=` | Original size |
| `zI` / `zO` / `z0` | Zoom in / out / original |
| `n=` | Zoom to size n |

## View

| Key | Action |
|-----|--------|
| `a` | Fit page (best-fit) |
| `s` | Fit width |
| `d` | Toggle dual-page |
| `D` | Cycle dual-page column |
| `r` | Rotate 90Â° |
| `^r` | Recolor (invert colors) |
| `F5` | Presentation mode |
| `F11` | Fullscreen |
| `^m` | Toggle input bar |
| `^n` | Toggle status bar |
| `Tab` | Toggle index (TOC) |

## Links

| Key | Action |
|-----|--------|
| `f` | Follow link (numbered hints) |
| `F` | Display link target |
| `c` | Copy link target |

## Quickmarks

| Key | Action |
|-----|--------|
| `mX` | Set quickmark X |
| `'X` | Go to quickmark X |

## Index Mode (Tab)

| Key | Action |
|-----|--------|
| `j` / `k` | Move down / up |
| `l` / `L` | Expand entry / expand all |
| `h` / `H` | Collapse entry / collapse all |
| `Enter` | Open entry |

## Mouse

| Input | Action |
|-------|--------|
| Scroll | Scroll up/down |
| `Ctrl+Scroll` | Zoom in/out |
| Middle-drag | Pan |
| Left-click | Follow link |
| Left-drag | Select text |

## Other

| Key | Action |
|-----|--------|
| `:` | Command mode |
| `o` / `O` | Open document |
| `R` | Reload |
| `q` | Quit |
| `Esc` / `^c` | Abort |




### Installation

```commandline
sudo apt-get install -y gh git
gh auth login
git clone https://github.com/MaxWolf-01/dotfiles.git
mv dotfiles .dotfiles
cd ~/.dotfiles
./install
```
*reboot*

Optional next steps:
- Use Makefile to install miniconda, docker, ...
